# lynxbox
A basic jquery plugin

Installation


1. Download

2. Create a div with "images" class and place images within
<dl>
<dt><div class="images">
          <img src='example.jpg' alt=''/>
                <img src='example.jpg' alt=''/></div></dt>
                </dl>
3. Customize

Call on the plugin and change theese values as you like.
```
$('images').lynxbox({
                borderColor:'blue',
                borderStyle:'solid',
                borderWidth:'1px'});
```
